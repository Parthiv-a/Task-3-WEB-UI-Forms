# Task 3: WEB UI Forms

This is a React 19 + TypeScript + Ant Design frontend for Task 3.
It connects to the backend (Task 1) and allows:
- List all tasks
- Search tasks by name
- Create / Update tasks
- Delete tasks
- Run a task and view output
- View task execution history in a modal
